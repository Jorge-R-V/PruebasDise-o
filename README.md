# 🎨 Pruebas Diseño

Bienvenido al repositorio experimental de diseño. Este espacio sirve como sandbox para explorar, probar y afinar componentes de interfaz de usuario, animaciones fluidas y layouts responsivos utilizando tecnologías modernas.

👉 **[Ver Aplicación en Vivo](https://Jorge-R-V.github.io/PruebasDise-o/)**

## 🛠️ Stack Tecnológico

- **React** - Librería principal para la construcción de interfaces.
- **TypeScript** - Tipado estático para un código robusto y seguro.
- **Tailwind CSS v4** - Framework de utilidades CSS para un estilado rápido y moderno.
- **Vite** - Entorno de desarrollo ultra rápido y empaquetador de módulos.

## 🚀 Instalación y Ejecución Local

Sigue estos pasos para clonar y ejecutar el proyecto en tu máquina local:

```bash
# 1. Clonar el repositorio
git clone https://github.com/Jorge-R-V/PruebasDise-o.git

# 2. Entrar al directorio
cd PruebasDise-o

# 3. Instalar las dependencias
npm install

# 4. Iniciar el servidor de desarrollo
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador para ver la aplicación en funcionamiento.
